# Brief
Interactive financial reporting dashboard created using Microsoft Excel. The model includes structured financial data, visual charts, and automated KPIs for better business decision-making.



# Project_Goal
- Provide a dynamic, interactive financial reporting dashboard
- Enable real-time tracking of revenue, expenses, and net profit
- Visualize monthly profit trends and expense breakdowns for data-driven financial decisions



# DataSet
- The dataset used in this project comprises structured financial records including monthly revenue, expenses, and category-based transaction data.



# How_It_Works
1. **Data Loading and Preparation**:  
   Financial data is imported and structured within Excel, utilizing Power Query for data cleaning and aggregation where necessary.

2. **Data Modeling and Analysis**:  
   Key metrics are calculated automatically:
   - Total Revenue, Total Expenses, and Net Profit are aggregated.
   - Monthly profit trends are derived from time-series financial data.
   - Expenses are broken down by category to identify cost drivers.

3. **Data Visualization**:  
   The dashboard generates insightful visualizations:
   - **KPI Cards**: Display Total Revenue, Total Expenses, and Net Profit at a glance.
   - **Monthly Profit Trend**: A line or column chart illustrating profit fluctuations over time.
   - **Revenue vs. Expense Analysis**: A comparative visualization highlighting financial performance.
   - **Category-Based Expense Breakdown**: A chart showing the percentage contribution of each expense category.

4. **Output**:  
   An interactive Excel dashboard providing comprehensive financial insights through real-time auto-calculated metrics and dynamic charts.
