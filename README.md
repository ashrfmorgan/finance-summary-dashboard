# finance-summary-dashboard
Finance Summary Dashboard built using Microsoft Excel, including dynamic KPI cards, monthly profit analysis, revenue vs. expense comparison, and category-based visualization. This project showcases financial reporting, data modeling, and performance tracking techniques suitable for business finance teams.



Interactive financial reporting dashboard created using Microsoft Excel.
The model includes structured financial data, visual charts, and automated KPIs for better business decision-making.

Key Features

• Monthly Profit Trend Visualization
• Revenue vs. Expense Analysis
• KPI Cards (Total Revenue – Total Expenses – Net Profit)
• Category-Based Expense Breakdown
• Easy-to-update Data Structure
• Real-time Auto-Calculated Metrics

Technologies Used

• Microsoft Excel
• Power Query (if used)
• Data Cleaning + Aggregation
• Pivot Charts + KPI Cards
• Financial Reporting & Visualization

Insights Uncovered

• Clear profit fluctuations month-to-month
• Expense category contribution %
• Revenue distribution patterns
• Identifying points of profit growth / decline
